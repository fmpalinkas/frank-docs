Additional classes that make development with Kotlin more pleasant and idiomatic. This module does
not add any new features to the existing NewMap API.

To use this library, add the following dependency to the build.gradle file:

```java
api("com.New.online:sdk-maps-ktx-extensions:{libversion}")
```

To apply an operation on the NewMap:

```java
inline fun MapFragment.applyOnNewMap(noinline block: NewMap.() -> Unit) {
    getAsyncMap(block)
}
```

To take advantage of coroutines:

```java
suspend inline fun MapFragment.getNewMap(): NewMap = suspendCoroutine { cont ->
    val callback = OnMapReadyCallback { NewMap -> cont.resume(NewMap) }
    getAsyncMap(callback)
}
```
