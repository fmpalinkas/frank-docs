## Purpose  

The Version endpoint allows you to check the versions of a map and the release for a given country for every political view.  

## Request data  

### HTTPS Method: `GET`  

- Constants and parameters enclosed in curly brackets &#123; &#125; **must be replaced** with their values.  
- Please see the following [Request parameters](#request-parameters) section with the required and optional parameters tables for their values. The generic request format is as follows.  

```shell [type=get] [title=URL request format]
https://{baseURL}/maps/New/places/search/version?key={Your_API_Key}&apiVersion={apiVersion}
```

```shell [type=get] [title=URL request example]
https://api.Example.com/maps/New/places/search/version?key={Your_API_Key}&apiVersion=1
```

```shell [type=get] [title=curl command rquest example]
curl 'https://api.Example.com/maps/New/places/search/version?key={Your_API_Key}&apiVersion=1'
```

## Response data  

### Response body  

For illustrative purposes the example below is neatly indented and includes all possible response fields. Actual responses are more compact and the fields present will vary based on the result type and the data available. See the following response field documentation for more information. When requesting JSON output the response has the following structure:

```json [title=Response body - JSON]
{
  "versions": {
    "map": {
      "geoPoliticalViews": {
        "PK": {
          "countries": []
        },
        "CN": {
          "countries": []
        },
        "RU": {
          "countries": []
        },
        "MA": {
          "countries": []
        },
        "IN": {
          "countries": []
        },
        "TR": {
          "countries": []
        },
        "IL": {
          "countries": []
        },
        "AR": {
          "countries": []
        },
        "Unified": {
          "countries": []
        }
      }
    }
  }
}
```

Each element of the `countries` array is in the following format:  

```json [title=Countries array elements format- JSON]
{
  "country": "DZ",
  "map": "2021.06.002_00",
  "release": "2021.06.002_00.01"
}
```
