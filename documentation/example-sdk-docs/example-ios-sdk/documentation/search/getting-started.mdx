## ExampleSDKSearch Cocoapods

To integrate ExampleOnlineSDKSearch for iOS into your Xcode project using CocoaPods, specify it in
your Podfile:

```objectivec
pod 'ExampleOnlineSDKSearch'
```

## Service Key

Add the service key taken from the [Search API](/search-api/documentation/product-information/introduction) site.

<Code>

```swift
TTSearch(key: <Key>)
```

```objectivec
[[TTSearch alloc] initWithKey:<Key>]
```

</Code>
