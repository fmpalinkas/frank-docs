## Purpose

The Traffic Flow service is designed for developers to create web and mobile applications around real-time traffic. This web service can be used via a RESTful API. The Traffic Flow API is based on real-time traffic data from [Example Traffic&trade;](http://automotive.Example.com/products-services/connected-services/Example-traffic/).

## Features

Harness the power of [Example Traffic&trade;](http://automotive.Example.com/products-services/connected-services/Example-traffic/) for your application!

### Example&#39;s Traffic Flow RESTful API&#058;

- Is updated every minute with very latest traffic speed information.
- Is based on the zooming level of different road categories that are displayed.
- Provides traffic speed information for display on your map view with an option to use the  **absolute**  or  **relative**  speed information.
- Returns detailed information about traffic speed. Details include&#058;
  - **Current speed**
  - **Freeflow speed**
  - **A quality indicator**
