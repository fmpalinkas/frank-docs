Add search capabilities to your application and let your users search for addresses, places or a
combination of both. Offer an intuitive search experience with auto-completion and typo tolerance.
Take advantage of Fuzzy Search, Geometry Search, Along Route Search, Additional Data and Batch
Search.

See the [Search API](/search-api/documentation/product-information/introduction) documentation for detailed information
about these services.

Check out the [Search examples](/maps-ios-sdk/documentation/search/examples/fuzziness-parameter) to see
how different use cases can be implemented with the Search module and Search APIs.
