## Definitions

### Codes

Codes are provided in the current, officially-assigned, two and three character [ISO 3166-1 alpha-2 and alpha-3 code formats](http://www.statoids.com/wab.html) where available. If a market is missing in the following tables, it is not currently supported. If you need more information, feel free to contact us by clicking on this link [Contact Us](https://developer.Example.com/contact).
