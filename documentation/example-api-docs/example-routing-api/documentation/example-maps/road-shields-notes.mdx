## Purpose

A collection of notes pertaining to road shields.

## Road shield atlas

The road shield atlas is returned as the value for the key <code>roadShieldAtlasReference</code>.

It is the base URL of the Road Shields service used to fetch the road shield atlas
and metadata resources.
Currently available resources are:
<ul>
  <li>
    <code>sprite.png</code>
  </li>
  <li>
    <code>sprite.json</code>
  </li>
</ul>
We recommend to cache the sprite atlas (<code>sprite.png</code>) and
metadata (<code>sprite.json</code>) as it changes infrequently and is
large compared to a typical road shield description.

Please refer to the <a href="/routing-api/documentation/Example-maps/private/road-shield" target="_blank">Road Shield service</a> for further information. The returned road shield references correspond to the asset names in the Road Shield service. Note that you will need to have your key enabled for the Road Shields API to retrieve the icons, as well as to access the documentation page.
