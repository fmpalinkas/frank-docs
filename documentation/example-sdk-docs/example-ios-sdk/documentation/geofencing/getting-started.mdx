## ExampleOnlineSDKGeofencing Cocoapods

To integrate ExampleOnlineSDKGeofencing for iOS into your Xcode project using CocoaPods, specify it
in your Podfile:

```objectivec
pod 'ExampleOnlineSDKGeofencing'
```

## Service Key

Add the service key taken from the [Geofencing API](/geofencing-api/documentation/product-information/introduction)
site.

<Code>

```swift
TTGeofencingReportService(key: <Key>)
```

```objectivec
[[TTGeofencingReportService alloc] initWithKey:<Key>]
```

</Code>
