## Definitions

### Address points

Derived from the Point Address index in the Search API. This field is only supported when Search and Geocoding are supported.

### House number

Derived from the Address Interpolation index in the Search API. This field is only supported when Search and Geocoding are supported.

### Codes

Codes are provided in the current, officially-assigned, two and three character [ISO 3166-1 alpha-2 and alpha-3](http://www.statoids.com/wab.html) code formats where available. If a market is missing in the following tables, it is not currently supported. If you need more information, please feel free to [**Contact Us**](https://developer.Example.com/contact-us).
