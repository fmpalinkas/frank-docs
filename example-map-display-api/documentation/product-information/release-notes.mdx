## Overview

This TomTom Release Notes page records the release date with changes and bug fixes associated with a product release. The release dates are in descending order starting with the most recent update. Map data is released twice a week.

### Release date: 2023.05.15

**Changes:**

- [Hillshade Tile API](/map-display-api/documentation/raster/hillshade-tile)
  - Added the Hillshade Tile API.

### Release date: 2023.04.26

**Changes:**

- [Vector Tile API](/map-display-api/documentation/vector/content-v2#places)
  - Added truck restrictions in V2 vector tiles. Please see the documentation.

### Release date: 2022.06.28  

**Changes:**  

- [Vector Tile API](/map-display-api/documentation/vector/content-v2#places)
  - Added house numbers in V2 vector tiles. Please see the documentation.

### Release date: 2022.05.17  

**Changes:**  

- [Vector Tile API](/map-display-api/documentation/vector/vector-service) 
  - Added version V2.
- [Map Styles](/map-display-api/documentation/mapstyles/map-styles-v2) 
  - Added version V2.

### Release date: 2022.05.09  

**Changes:**  

- [Vector Tile API](/map-display-api/documentation/vector/vector-service) 
  - Added support for South Korea with detailed map.
- [Raster Tile API](/map-display-api/documentation/raster/raster-service) 
  - Added support for South Korea with detailed map.

### Release date: 2022.04.19  

**Changes:**  

- [Satellite Tile API](/map-display-api/documentation/raster/satellite-tile)  
  - Added the Satellite Tile API.


### Release date: 2018.06.26  

**Changes:**  

- [Vector Tile API](/map-display-api/documentation/vector/vector-service) 
  - Added additional zoom levels.

### Release date: 2018.04.25  

**Changes:**  

- [Raster Tile API](/map-display-api/documentation/raster/raster-service) 
  - Added the Language parameter. Please see the documentation.
  - Updated a new style and layout for raster tiles.

### Release date: 2017.10.06  

**Changes:**  

- [Raster Tile API](/map-display-api/documentation/raster/raster-service) 
  - Added the View parameter. Please see the documentation.

### Release date: 2017.02.16  

**Changes:**  

- [Vector Tile API](/map-display-api/documentation/vector/vector-service) 
  - Published the Online Maps Vector service (map version 2016.09).

### Release date: 2015.05.21  

**Changes:**  

- [Raster Tile API](/map-display-api/documentation/raster/raster-service) 
  - Added additional map styles. Please see the Documentation.

### Release date: 2015.03.31  

**Changes:**  

- [Raster Tile API](/map-display-api/documentation/raster/raster-service) 
  - Updated toll roads visualization.
  - Updated road shields dedicated per country.
  - Updated the signed national parks.
