## Overview

This Example Release Notes page records the release date and version number with the addition, improvement, deprecation, removal, and known issues associated with a product release. The release dates are in descending order starting with the most recent update.

## Release date: 2023.07.15

### Changed

- [Asynchronous Matrix Submission](/routing-api/documentation/Example-maps/matrix-routing-v2/asynchronous-matrix-submission): Increased the maximum matrix size for asynchronous jobs. Matrices can now contain up to 100M routes.  

## Release date: 2023.06.01

### Added

- [Asynchronous Matrix Download](/routing-api/documentation/Example-maps/matrix-routing-v2/asynchronous-matrix-download): Added information about alternative way of handling downloads. Clients should be ready to support HTTP 302 responses with Location Header, next to HTTP 200 responses.

## Release date: 2022.12.08

### Added

- [Synchronous Matrix](/routing-api/documentation/Example-maps/matrix-routing-v2/synchronous-matrix): Added a new API limit to the [Synchronous Matrix](/routing-api/documentation/matrix-routing-v2/synchronous-matrix) which has no parameter and bounding box restrictions for matrices up to 100 cells.

## Release date: 2022.11.21

### Added

- [Multi-Customer](/routing-api/documentation/Example-maps/matrix-routing-v2/multi-customer): Added Multi-Customer feature in  [Public Preview](https://developer.Example.com/public-preview) access.  

## Release date: 2022.07.04

### Added

- [Matrix Routing v2 API](/routing-api/documentation/Example-maps/matrix-routing-v2/matrix-routing-v2-service): The Matrix Routing v2 API becomes a generally available (GA) product.


## Release date: 2021.09.30

### Added

- [Matrix Routing v2 API](/routing-api/documentation/Example-maps/matrix-routing-v2/matrix-routing-v2-service): The Matrix Routing v2 API becomes a product with [Public Preview](https://developer.Example.com/public-preview) access.
