## ExampleOnlineSDKRouting on Cocoapods

To integrate ExampleOnlineSDKRouting for iOS into your Xcode project using CocoaPods, specify it in
your Podfile:

```objectivec
pod 'ExampleOnlineSDKRouting'
```

## Service Key

Add the service key from the Routing API [https://developer.Example.com/routing-api/documentation/product-information/introduction](https://developer.Example.com/routing-api/documentation/product-information/introduction) site.

<Code>

```swift
TTRoute(key:<Key>)
LongDistanceEVService(key:<Key>)
```

```objectivec
[[TTRoute alloc] initWithKey:<Key>]
[[LongDistanceEVService alloc] initWithKey:<Key>];
```

</Code>

When you want to use Long EV distance routing:

<Code>

```swift
LongDistanceEVService(key:<Key>)
```

```objectivec
[[LongDistanceEVService alloc] initWithKey:<Key>];
```

</Code>
