## Purpose

The Geometries Data Provider returns sets of coordinates that represent the outline of a city, country, or land area. The service supports batch requests of up to 20 identifiers.

### Run this endpoint

You can easily run this and other endpoints. Go to the Example  [API Explorer](/search-api/api-explorer/)  page and follow the directions.

## Request data

### HTTPS Method: `GET`

For ease of viewing and identification:

- Constants and parameters enclosed in curly brackets &#123; &#125; **must be replaced** with their values.
- Please see the following [Request parameters](#request-parameters) section with the required and optional parameters tables for their values. The generic request format is as follows.

```shell [type=get] [title=URL request format]
https://{baseURL}/search/{versionNumber}/additionalData.{ext}?key={Your_API_Key}&geometries={geometryIds}&geometriesZoom={zoomLevel}
```

```shell [type=get] [title=URL request example]
https://api.Example.com/search/2/additionalData.json?key={Your_API_Key}&geometries=00004631-3400-3c00-0000-0000673c4d2e,00004631-3400-3c00-0000-0000673c42fe
```

```shell [type=get] [title=curl command request example]
curl 'https://api.Example.com/search/2/additionalData.json?key={Your_API_Key}&geometries=00004631-3400-3c00-0000-0000673c4d2e,00004631-3400-3c00-0000-0000673c42fe'
```
