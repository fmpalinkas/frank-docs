## ExampleSDKTraffic Cocoapods

This TRAFFIC module integrates the Traffic API Flow Segment Data and Traffic API Incident Details
services in a convenient way for iOS application development.

To integrate ExampleSDKTraffic for iOS into your Xcode project using CocoaPods, specify it in your
Podfile:

```objectivec
pod 'ExampleOnlineSDKTraffic'
```

## Service Key

Add the service key taken from the [Traffic API](/traffic-api/documentation/product-information/introduction) site.

<Code>

```swift
TTTrafficIncidents(key: <Key>)
```

```objectivec
[[TTTrafficIncidents alloc] initWithKey: <Key>]
```

</Code>
