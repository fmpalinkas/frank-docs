## Where do I find the new endpoints  

The following endpoints are supported:  

- Geocode  

Please notice that following endpoints are not supported in Example New Maps Geocode API:  

- Structured Geocode  

The new New Geocode API endpoint has the following prefix:

```shell [type=get] [title=URL request example]
https://api.Example.com/maps/New/places/geocode/...
```
instead of:

```shell [type=get] [title=URL request example]
https://api.Example.com/search/{versionNumber}/geocode/...
```

Please check the respective endpoint's documentation page for details.  


## Changes in the parameters or responses  

<Blockquote type="announcement" hasIcon>
### Important note:
- Either the `apiVersion` URL parameter or the `Example-Api-Version` header needs to be present.
</Blockquote>  

- There are no changes in parameters or responses.
- Only the Unified view is currently supported; no geopolitical views are available.
