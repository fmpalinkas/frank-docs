  name="rx-extensions"
></a>

Additional classes that allow you to write reactive components using NewMap. Specific map
callbacks (e.g., map click listener) are transformed to Observables so they can be easily integrated
with RxJava2 types. This module does not add any new features to the existing NewMap API.

To use this library, add the following dependency to the build.gradle file:

```java
api("com.New.online:sdk-maps-rx-extensions:{libversion}")
```

To initialize the Rx wrapper for NewMap:

```java
RxNewMap rxNewMap = new RxNewMap(NewMap)
```

To subscribe for map click events:

```java
Disposable mapClickDisposable = rxNewMap
        .getOnMapClickObservable()
        .observeOn(mainThread())
        .subscribe(latLng -> displayMessage(
                R.string.menu_events_on_map_click,
                latLng
        ));
```
